# Credit Scoring Model

## Overview
This project predicts an individual's credit score category (Low, Average, High) using machine learning techniques.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NumPy

## Algorithm
- Random Forest Classifier
** Results**
- Accuracy: 96.97%
- Precision, Recall, F1-score evaluation
- Confusion Matrix analysis
- Feature Importance analysis

## Dataset Features
- Age
- Gender
- Income
- Education
- Marital Status
- Number of Children
- Home Ownership
